Test application with IBGE API
Data search regarding registration of names every 10 years from 1930 onwards
Docker PHP Apache
comandos docker

criar imagem :
       docker build -t {NOME IMAGEM} .

execultar : 
       docker run -it -p 8000:80 -v {PATH LOCAL }/:/var/www/html/ {NOME IMAGEM}

navegador 
localhost:8000
